# 🌸 Vivi Boty 🧪✨
Status do Projeto: 🚀 Concluído / Em constante evolução

Link para a vitrine: [Acesse a Vivi Boty aqui!](https://vivi-boty.netlify.app/)

💖 Uma nota especial para a minha Mãe: O que é a Vivi Boty?
Mãe, a Vivi Boty é o seu novo espaço na internet para o seu negócio de perfumes! Eu criei esse site para ser a sua vitrine virtual. Agora, os seus clientes podem ver todos os seus perfumes, conferir os preços, aproveitar as promoções e, com apenas um clique, eles já são direcionados para falar com você e fechar a compra. Fiz com muito carinho para te ajudar a vender mais, alcançar novos clientes e facilitar o seu atendimento no dia a dia!

📌 Sobre o Projeto
A Vivi Boty é uma plataforma web de catálogo digital e e-commerce simplificado desenvolvida especificamente para pequenos empreendedores (neste caso, focado no nicho de perfumaria). O projeto nasceu da necessidade real de digitalizar um negócio físico, tornando o processo de escolha do produto visual, informativo e integrado.

Para o meu portfólio, este projeto demonstra competência em:

UI/UX focado em Conversão: Layout limpo e atraente que destaca fotos, preços e promoções para prender a atenção do cliente.

Funil de Vendas Direto (Click-to-Chat): Integração com APIs de comunicação para reduzir a fricção entre o interesse do cliente e a finalização da compra.

Acessibilidade e Usabilidade: Uma interface extremamente simples de navegar, pensada tanto para o público comprador quanto para a facilidade de gestão do negócio.

🛠️ Tecnologias Utilizadas
HTML5 & CSS3: Estruturação semântica e estilização moderna, com uso de componentes visuais atraentes para destacar os perfumes e as etiquetas de promoção.

JavaScript (ES6+): Lógica para filtragem de produtos, exibição dinâmica de valores e automação do link de contato (gerando a mensagem certa com o produto escolhido).

Netlify: Hospedagem rápida e otimizada para garantir que a vitrine carregue instantaneamente no celular dos clientes.

✨ Funcionalidades Principais
🛍️ Vitrine Digital de Perfumes: Organização visual dos produtos com fotos, descrições detalhadas e volumetria.

🏷️ Destaque de Preços e Promoções: Sinalização clara de descontos para incentivar a compra.

📲 Botão "Adquirir" Inteligente: O cliente escolhe o perfume e, ao clicar, abre o WhatsApp com uma mensagem prontinha (ex: "Olá, gostei do perfume [Nome] e quero encomendar!"), agilizando o atendimento da empreendedora.

📱 Mobile-First/Responsivo: Totalmente otimizado para celulares, que é por onde a maioria dos clientes acessa as redes sociais e links de compras.

👩‍💻 Como um cliente usa o site?
Ele acessa o link: vivi-boty.netlify.app

Navega pelos perfumes disponíveis e vê os valores e promoções.

Escolhe o seu perfume favorito e clica no botão para comprar/entrar em contato.

O site abre o WhatsApp diretamente na conversa com a sua mãe, já dizendo qual produto ele tem interesse. Pronto, venda facilitada!
